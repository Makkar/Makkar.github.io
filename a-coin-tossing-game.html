The Game
I recently came across this simple to state puzzle:
Suppose you have a coin that has probabilities ppp﻿ for heads and 1−p1-p1−p﻿ for tails. You play the following game with a friend who also knows the value of p.p.p.﻿ The first player picks one of the outcomes HHHHHH﻿, THTHTH﻿, HTHTHT﻿ or TTTTTT﻿. The second player observes the choice of the first player and picks one of the remaining 3 outcomes. You then proceed to toss the coin infinitely many times independently constructing an infinite sequence of HHH﻿’s and TTT﻿’s. A player wins if their choice appears first in the sequence. For example, if player one chose HHHHHH﻿ and player two chose TTTTTT﻿ and if the coin toss sequence observed is HTHTHTTHTHHH...HTHTHTTHTHHH...HTHTHTTHTHHH...﻿, then it results in a victory for player two since TTTTTT﻿ occurred before HHHHHH﻿ in this sequence. When is it better to go first in this game?
Let's make the conditions under which a player wins more concrete.
The strategy: It’s better to go as the first player in the game if for some choice a∈{HH,TT,HT,TH}a \in \{HH,TT,HT,TH\}a∈{HH,TT,HT,TH}﻿, the probability of winning the game is greater than 1/21/21/2﻿ no matter the choice of the second player. It’s better to go as the second player in the game if for every choice aaa﻿ of the first player, we can find some element of the set {HH,TT,HT,TH}∖{a}\{HH,TT,HT,TH\} \setminus \{a\}{HH,TT,HT,TH}∖{a}﻿, such that the probability of winning the game is greater than 1/2.1/2.1/2.﻿
As an example, suppose p=3/4p = 3/4p=3/4﻿, then if the first player chooses HHHHHH﻿, then no matter the choice of the second player, the probability of the first player winning the game is greater than 1/2.1/2.1/2.﻿ To see this note that the probability of getting two consecutive heads in the first two coin tosses itself is 9/16>1/2.9/16 > 1/2.9/16>1/2.﻿ Therefore, for p=3/4p=3/4p=3/4﻿ it is better to go as the first player.
On first impression it seems as if it must be better to go as the first player, no matter the value of ppp﻿, since the first player has more choices, but as we will see, the fact that the second player has the advantage of choosing the outcome after observing the choice of the first player gives him an advantage for certain values of p.p.p.﻿
But before we do that, we should prove an implicit assumption: the game ends with a winner in a finite number of moves with probability 1.1.1.﻿ But to do that we first need to define a probability space on which the game is played. In particular, does it even exist? Indeed it does as we see below.
Constructing the probability space
To formalize our analysis we need to define a sequence of i.i.d. random variables, one for each coin toss. However, the existence of a probability space on which we can define this sequence is not obvious at all. Back to the game
Recall the strategy outlined above. Without loss of generality we may assume p≥1/2p \geq 1/2p≥1/2﻿ because otherwise we can just flip the tags HHH﻿ and TTT﻿. From a first player perspective we just want to find one of the dominant choices HHHHHH﻿, THTHTH﻿, HTHTHT﻿ or TTTTTT﻿. Let's calculate the minimum ppp﻿ we get for each choice.
If player 111﻿ chooses HHHHHH﻿ and player 222﻿ chooses THTHTH﻿, then it's better to be player 111﻿ if p>1/2.p > 1/\sqrt{2}.p>1/2​.﻿ Checking for other choices of player 222﻿ we see that THTHTH﻿ is the optimal choice.
