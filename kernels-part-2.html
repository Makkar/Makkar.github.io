Introduction
We discussed some functional analysis and the basics of Reproducing Kernel Hilbert Space theory in a previous two blog posts. The aim of this article is to discuss the need for approximating the kernel matrix and one method in particular to do that, Random Fourier Features.
Let's start by discussing Kernel Ridge Regression, a simple application of kernel methods.
Kernel Ridge Regression
Ridge Regression
Let's recall the ridge regression model. We have some training data of the form:
(x1,y1),…,(xn,yn)∈(X,Y),(x_1, y_1), \ldots, (x_n, y_n) \in (\mathcal{X}, \mathcal{Y}),(x1​,y1​),…,(xn​,yn​)∈(X,Y),

Kernel Ridge Regression
The previous model suffers from limited expressiveness. As we have seen before, the idea of kernel methods is to define a map which takes our inputs from X\mathcal{X}X﻿ to an RKHS H\mathcal{H}H﻿,
ϕ:X→H\phi : \mathcal{X} \to \mathcal{H}ϕ:X→H
and then apply the linear model of ridge regression above to ϕ(xi)\phi(x_i)ϕ(xi​)﻿ instead of xi.x_i.xi​.﻿ H\mathcal{H}H﻿ could be an infinite-dimensional vector space, but for now suppose it is DDD﻿-dimensional. Let Φ\mathbf{\Phi}Φ﻿ represent the n×Dn \times Dn×D﻿ matrix whose ithi^{\text{th}}ith﻿ row is ϕ(xi)⊤.\phi(x_i)^\top.ϕ(xi​)⊤.﻿ Then the ridge regression solution is given by
wRR=(λσ2ID+Φ⊤Φ)−1Φ⊤y.\mathbf{w}_{\text{RR}} = (\lambda \sigma^2 \mathbf{I}_D + \mathbf{\Phi}^\top \mathbf{\Phi})^{-1} \mathbf{\Phi}^\top \mathbf{y}.wRR​=(λσ2ID​+Φ⊤Φ)−1Φ⊤y.

wRR=Φ⊤(λσ2In+ΦΦ⊤)−1y.\mathbf{w}_{\text{RR}} = \mathbf{\Phi}^\top (\lambda \sigma^2 \mathbf{I}_n + \mathbf{\Phi} \mathbf{\Phi}^\top)^{-1} \mathbf{y}.wRR​=Φ⊤(λσ2In​+ΦΦ⊤)−1y.

and prediction for a test data point, xxx﻿, is

where kxik_{x_i}kxi​​﻿ denotes the reproducing kernel for the point xi.x_i.xi​.﻿
If this looks suspiciously similar to the Representer theorem (Theorem-8), then that's because it is an instance of it! So prediction takes O(n)\mathcal{O}(n)O(n)﻿ time.
The point of this discussion was to show that Kernel Ridge Regression is computationally expensive. Simply storing the gram matrix, K\mathbf{K}K﻿, takes O(n2)\mathcal{O}(n^2)O(n2)﻿ space. We need to find methods which can circumvent using K\mathbf{K}K﻿ if we are to apply kernel methods to anything other than the smallest of the data sets.
Kernel Approximation
The idea behind kernel approximation methods is to replace the gram matrix, K\mathbf{K}K﻿, with a low rank approximation, i.e., find an n×Sn \times Sn×S﻿ matrix, Z\mathbf{Z}Z﻿, such that
K≈ZZ⊤.\mathbf{K} \approx \mathbf{Z} \mathbf{Z}^\top.K≈ZZ⊤.
We want S≪nS \ll nS≪n﻿, and thus we significantly reduce our time and space complexity. Z\mathbf{Z}Z﻿ takes only O(nS)\mathcal{O}(nS)O(nS)﻿ space. Inversion of (λσ2In+ZZ⊤)(\lambda \sigma^2 \mathbf{I}_n + \mathbf{Z} \mathbf{Z}^\top)(λσ2In​+ZZ⊤)﻿ takes only O(nS2)\mathcal{O}(n S^2)O(nS2)﻿ time.
One way to think about this is we want to find a mapping of xix_ixi​﻿'s to some latent space such that the inner product in this latent space approximates the kernel
Ki,j=K(xi,xj)≈zi⊤zj,\mathbf{K}_{i,j} = K(x_i, x_j) \approx z_i^\top z_j,Ki,j​=K(xi​,xj​)≈zi⊤​zj​,
where ziz_izi​﻿ denotes the ithi^\text{th}ith﻿ row of Z.\mathbf{Z}.Z.﻿ One popular way to do that is by Nyström approximation, which I won't be discussing at all. The other popular method is to use random Fourier features. This approach was introduced by Rahimi and Recht in their seminal 2007 paper Random Features for Large-Scale Kernel Machines. It relies on a result from functional analysis called Bochner's theorem, so let's digress and discuss that first.
Bochner’s Theorem
Definition 28: A function K:Rn→CK : \R^n \to \mathbb{C}K:Rn→C﻿, is said to be positive-definite if
∑i,j=1nαiα‾jK(xi−xj)≥0\sum _{i,j = 1}^n \alpha_i \overline{\alpha}_j K(x_i - x_j) \geq 0i,j=1∑n​αi​αj​K(xi​−xj​)≥0

The following theorem is the converse of Bochner's theorem. We state it first since it is easier to prove.
Theorem 9 [Converse of Bochner's theorem]: The Fourier transform of every finite Borel measure on Rn\R^nRn﻿ is positive-definite.

Random Fourier Features
Picking up where we left: We want to find a low-dimensional mapping of xix_ixi​﻿'s into a latent space such that we can approximate the kernel computation with an inner product in this latent space. We will use Bochner's theorem for this. To apply this theorem we need to limit ourselves to a special class of kernels.
Definition 29: A kernel function K:Rd×Rd→CK : \R^d \times \R^d \to \mathbb{C}K:Rd×Rd→C﻿ is called shift-invariant if K(x,y)=K(x−y,0).K(x, y) = K(x-y, 0).K(x,y)=K(x−y,0).﻿
Gaussian and Laplacian kernels are examples of shift-invariant kernels.
Note that the function K′:Rd→CK': \R^d \to \mathbb{C}K′:Rd→C﻿ defined by K′(x)=K(x,0)K'(x) = K(x, 0)K′(x)=K(x,0)﻿ is positive-definite if KKK﻿ is a shift-invariant kernel. Bochner's theorem then implies the existence of a spectral measure, μ\muμ﻿, such that
K(x,y)=K(x−y,0)=K′(x−y)=∫Rdexp⁡{−iω⊤(x−y)} dμ(ω).K(x, y) = K(x-y, 0) = K'(x-y) = \int _{\R^d} \exp\{-i\omega^\top (x-y)\} \,\mathrm{d}\mu(\omega).K(x,y)=K(x−y,0)=K′(x−y)=∫Rd​exp{−iω⊤(x−y)}dμ(ω).
Let ppp﻿ denote the Radon-Nikodym derivative of μ\muμ﻿ with respect to the Lebesgue measure on Rd\R^dRd﻿ (or more simply, the density of μ\muμ﻿), then we can write the equation above as
K(x,y)=∫Rdexp⁡{−iω⊤(x−y)}p(ω) dω(⋆)K(x, y) = \int _{\R^d} \exp\{-i\omega^\top (x-y)\} p(\omega) \,\mathrm{d}\omega \tag{$\star$}K(x,y)=∫Rd​exp{−iω⊤(x−y)}p(ω)dω(⋆)
For example, if we take the Gaussian kernel
K(x,y)=exp⁡(−∥x−y∥222σ2),K(x,y) = \exp\left(-\frac{\lVert x-y\rVert^2_2}{2 \sigma^2}\right),K(x,y)=exp(−2σ2∥x−y∥22​​),
then since K(0,0)=1K(0,0) = 1K(0,0)=1﻿, ppp﻿ is a probability density, and can be easily shown to be Gaussian
p∼N(0,1σ2Id).p \sim \mathcal{N}\left(0, \frac{1}{\sigma^2} \mathbf{I}_d\right).p∼N(0,σ21​Id​).
We now see an obvious way to approximate the kernel from equation (⋆)(\star)(⋆)﻿: use Monte Carlo approximation. If we take SSS﻿ samples, ω1,…,ωS∼p(ω)\omega_1, \ldots, \omega_S \sim p(\omega)ω1​,…,ωS​∼p(ω)﻿,

where z:Rd→RSz : \R^d \to \R^Sz:Rd→RS﻿ is the latent mapping given by
z(x)=1S[exp⁡(−iω1⊤x),…,exp⁡(−iωS⊤x)]⊤.z(x) = \frac{1}{\sqrt{S}}[\exp(-i \omega ^\top _1 x), \ldots, \exp(-i \omega ^\top _S x)]^\top.z(x)=S​1​[exp(−iω1⊤​x),…,exp(−iωS⊤​x)]⊤.

There exist many modifications to the basic RFF approach described here. For example, we can do Quasi-Monte Carlo approximations instead of Monte Carlo approximations. See the paper Quasi-Monte Carlo Feature Maps for Shift-Invariant Kernels by Avron, Sindhwani, Yang and Mahoney (2016). Or we could sample from a modified distribution in Fourier space, given by the leverage function of the kernel. See the paper Random Fourier Features for Kernel Ridge Regression: Approximation Bounds and Statistical Guarantees by Avron et al (2017).

Epilogue
Random Fourier Features is an easy to implement approach that allows us to apply kernel methods on large data sets. I haven't discussed the number of samples, SSS﻿, needed to approximate the kernel function within an error bound. You can find such results in the papers linked above.
